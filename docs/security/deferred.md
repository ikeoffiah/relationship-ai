# Security Items Deferred to Post-Pilot

## External Penetration Test
- **Trigger**: Before public launch or when onboarding 100+ users.
- **Cost**: $5k–15k (engage firm with healthcare app experience).
- **Current mitigation**: OWASP ZAP weekly scans + namespace isolation CI gates.

## Formal Threat Model (STRIDE)
- **Trigger**: Before public launch.
- **Current mitigation**: Internal review with developers + clinical advisor.

## Bug Bounty Program
- **Trigger**: After external pentest at scale.
